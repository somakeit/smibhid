SMIBHID - Sensors

SMIBHID can have multiple sensors attached for measuring the environment and reporting these values over various API endpoints..

SCD30

The SCD30 sensor module comprises temperature, humidity and high accuracy CO2 sensor. The sensor can be calibrated by placing in a known CO2 concentration (such as outdoors) and send the calbration command with the current CO2 PPM value.

The calibration can be performed use the API, or using the SCD30 control page.
